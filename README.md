# Swiggy Dashboard Analysis

## Overview

This project is a Power BI dashboard created to analyze Swiggy food delivery data and generate business insights using interactive visualizations.

The dashboard helps in understanding:

* Order trends
* Revenue performance
* Customer behavior
* Restaurant performance
* Delivery analysis

---

## Tools & Technologies Used

* Power BI
* DAX
* Excel / CSV
* Data Cleaning
* Data Visualization

---

## Dashboard Features

* Interactive filters and slicers
* KPI cards for quick insights
* Restaurant-wise analysis
* Order trend analysis
* Revenue and sales tracking
* Category-wise performance
* City-wise comparison

---

## KPIs Included

* Total Orders
* Total Revenue
* Average Order Value
* Top Restaurants
* Customer Ratings
* Delivery Performance
* Peak Order Hours

---

## Key Insights

* Identified peak ordering periods
* Found top-performing restaurants based on revenue and orders
* Compared city-wise sales performance
* Analyzed customer ordering patterns
* Evaluated delivery efficiency and ratings

1. Download the `.pbix` file.
2. Open it using Power BI Desktop.
3. Refresh the dataset if required.
4. Explore the dashboard using filters and slicers.
